API de Restaurante com Spring Boot
projeto acadêmico desenvolvido para a disciplina de Programar em Linguagem Orientada a Objetos avançada:
  - tecnologias:
java21
spring boot
spring web
spring Data jpa
MySQL
Maven

  - Funcionalidades
CRUD de clientes
CRUD de pedidos
Relacionamento entre Cliente e Pedido
Persistência em banco de dados relacional MySQL
Testes via Postman

  - Como executar
Criar o banco 'restaurante_db' no MySQL
configurar o 'application.properties'
Rodar:
cmd
mvnw.cmd spring-boot:run
